# Retrieva::Cop

Retrieva::Cop is a RuboCop configuration gem for Retrieva, inc.
Highly inspiered by [onkcop](https://github.com/onk/onkcop/) and [deka_eiwakun](https://github.com/esminc/deka_eiwakun/)
